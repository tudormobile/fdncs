## FinancialData Library Namespaces
The namespaces available in the *Tudormobile.FinancialData* library are described below. In several cases, including the namespace is required to expose extension methods provided.

### Tudormobile.FinancialData.dll

```
using Tudormobile.FinancialData;
using Tudormobile.FinancialData.Extensions;
```

- [Tudormobile.FinancialData](Tudormobile.FinancialData.yml)
    - Root namespace for the library. Essential classes that comprise the api.
- [Tudormobile.FinancialData.Extensions](Tudormobile.FinancialData.Extensions.yml)
    - Extensions methods.
